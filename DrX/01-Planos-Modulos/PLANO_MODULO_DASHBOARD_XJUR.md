# Plano do Modulo Dashboard do XJUR

## Missao do Modulo

`Dashboard` existe para transformar dados dispersos do sistema em visao executiva e acao imediata.

## Papel Estrategico

Ele nao deve ser mural decorativo.
Deve orientar prioridade, risco e proximo passo.

## Verdades do Modulo

- dashboard nao e fonte primaria de dados
- deve refletir trabalho real
- indicador sem acao perde valor
- cada card precisa responder "o que fazer agora"

## Escopo Nucleo

- indicadores executivos
- alertas operacionais
- atalhos para acao
- visao consolidada por tenant

## Riscos Atuais

- virar painel bonito sem utilidade pratica
- exibir metricas sem prioridade
- misturar informacao operacional e gerencial sem hierarquia

## Direcao de Evolucao

- menos quantidade, mais relevancia
- indicadores vinculados a acao
- recortes por perfil de uso
